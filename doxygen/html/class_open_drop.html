void OpenDrop::update ( void  )

The update-function should be called in the main loop of the programme.

  1. Clear the Fluxls Array
  2. Fill the Fluxls Array with the new data
  3. If the feedback feature is enabled: update the pad_feedback array
  4. Fill the chip with Fluxls array data
  5. Update the display

void OpenDrop::update_Drops ( void  )

Every droplet on the electrode matrix is being checked for updates. If the position has been changed, the new positions will be adopted.

  1. Check if feedback feature is enabled
  2. Check if the position has been changed
  3. Check if feedback has detected a change
  4. Call drop.begin()
